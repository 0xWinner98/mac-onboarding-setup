# 航海家 · AI 工具一键安装 (macOS)

打开「终端」，粘贴运行：

```bash
curl -fsSL https://raw.githubusercontent.com/0xWinner98/mac-onboarding-setup/main/install.sh | bash
```

自动检测并安装 Claude Code / Codex / Hermes / 飞书 CLI / Obsidian，**已装好的自动跳过**，再按顺序带你登录授权。

- 只检测不安装：命令末尾加 `| bash -s -- --check`
- 工具均从各家**官方源**安装，脚本内**不含任何密钥**
- Claude Code 怕封号 / 没订阅：登录时选「中转」，按提示装 CC Switch，中转配置找小组长拿

卡住就截图发小组长。
